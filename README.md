# Versionskontrolle Vereinfachen

Willkommen bei **Versionskontrolle Vereinfachen**! Diese Software zielt darauf ab, die Nutzung von Versionskontrollsystemen für Anfänger und kleine Teams zu erleichtern.

## Funktionen
- **Intuitive Benutzeroberfläche**: Ermöglicht einfaches Navigieren durch verschiedene Versionen von Dateien.
- **Einfache Befehle**: Verwaltet Versionen ohne komplexe CLI-Befehle.
- **Teamkollaboration**: Unterstützt grundlegende Funktionen zur Zusammenarbeit, wie Pull-Requests und Merge-Resolver.
- **Dateiversionsverlauf**: Zeigt eine klare Historie von Änderungen in Dokumenten.

## Installation
1. Klone dieses Repository.
2. Installiere die Abhängigkeiten mit `npm install`.
3. Starte die Anwendung mit `npm start`.
